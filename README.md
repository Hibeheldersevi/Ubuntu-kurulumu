# Ubuntu-kurulumu
## kurmak için gerekenler:
##### 1. Bilgisayarın şarjı %100 olsun.
##### 2. Bilgisayarımızın bir boş yerel Disk olmalı(ubuntuyu oraya kaydetmek için).
##### 3. 12Gb >= bir flashımız olmalı(boş olsa daha iyi çünkü içindeki herşey silinecek, yada içindekilerin yedeği alırsınız)
